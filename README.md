## Steps to Setup (with Docker)

<br />

**1. Clone repository**

```bash
git clone https://github.com/khylko98/black-knight-react-fastapi.git
```

<br />

**2. Run app using docker**

```bash
docker compose up
```

<br />

**3. Go by http://localhost:5173 and you will see next page**
